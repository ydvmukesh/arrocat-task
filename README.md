# Experience Highlights - Next.js

This is a Next.js conversion of the Experience Highlights scroll animation project, featuring GSAP and ScrollTrigger animations.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Scroll-based animations using GSAP and ScrollTrigger
- Responsive design
- Three animated slides with images and content
- Smooth transitions and transformations

## Technologies Used

- Next.js 14
- React 18
- TypeScript
- GSAP 3.12.5
- CSS Variables for dynamic styling

## Build

To create a production build:

```bash
npm run build
npm start
```
